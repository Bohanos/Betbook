BetBook

A secure, full-stack betting and booking application built with React, Vite, Tailwind CSS, and Python/FastAPI.

📖 About the Project
BetBook is a project-in-progress designed to provide a secure and intuitive platform for users to manage sports bookings and payouts. As a software engineering student, I am using this project to master full-stack architecture, including:

Real-time UI state management with React.

Scalable backend integration with FastAPI.

Responsive, accessible interface design using Tailwind CSS.

🚀 The Journey (Building in Public)
This project is currently under active development. My goal is to bridge the gap between complex backend systems and user-centric frontend experiences.

Phase 1: Foundation - Project initialization with Vite, routing setup, and global design system.

Phase 2: Authentication - Secure user registration and session handling.

Phase 3: The Betting Engine - Interactive game lists, odds calculation, and live tickers.

Phase 4: Finance & Administration - Wallet management and secure admin control panels.

## 💡 Why BetBook?
I initiated this project to challenge myself with the complexities of state-heavy applications. The primary goals are:
* Mastering the **React Router DOM** and **state management** lifecycle.
* Learning how to handle **asynchronous data** from a backend API in a real-world scenario.
* Building a **responsive, accessible design** system from scratch using Tailwind CSS.
